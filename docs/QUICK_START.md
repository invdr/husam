# Быстрый старт: Настройка каталога с Google Forms

> Архив: текущий каталог работает через PocketBase и `/admin`, а не через Google Forms/Sheets. Для актуального входа см. `CURRENT_HANDOFF.md`, `ARCHITECTURE.md` и `DEVELOPMENT.md`.

## 🚀 Краткая инструкция

### 1. Настрой Google сервисы

Следуй подробной инструкции в [GOOGLE_SETUP.md](./GOOGLE_SETUP.md)

**Кратко:**

1. Создай Google Sheets таблицу с заголовками
2. Создай Google Form и свяжи с таблицей
3. Настрой Apps Script для обработки фото
4. Скопируй ID таблицы

### 2. Настрой переменные окружения

1. Скопируй `.env.example` в `.env`:

   ```bash
   cp .env.example .env
   ```

2. Открой `.env` и заполни:

   ```
   VITE_GOOGLE_SHEET_ID=твой_id_таблицы
   VITE_GOOGLE_SHEET_NAME=Portfolio
   ```

3. ID таблицы можно найти в URL:
   ```
   https://docs.google.com/spreadsheets/d/ТВОЙ_ID/edit
   ```

### 3. Запусти проект

```bash
npm install
npm run dev
```

### 4. Проверь работу

1. Открой сайт в браузере
2. Перейди на секцию "Портфолио"
3. Убедись, что карточки загружаются из Google Sheets

### 5. Дай ссылку заказчику

Отправь заказчику ссылку на Google Form — он сможет добавлять карточки самостоятельно.

---

## ⚠️ Важно

- **Не коммить `.env` в Git** — он уже в `.gitignore`
- **Проверь, что таблица опубликована** для чтения
- **Протестируй форму** перед тем, как давать ссылку заказчику
