# Пошаговая настройка Google Forms + Google Sheets + Google Drive + Apps Script

## 📋 Общая схема работы

1. **Google Forms** — форма для заказчика (добавление карточек)
2. **Google Sheets** — хранилище данных каталога
3. **Apps Script** — автоматическая обработка данных формы
4. **Сайт** — читает данные из Google Sheets и показывает каталог

---

## 🚀 ШАГ 1: Создание Google Sheets таблицы

### 1.1. Создай новую таблицу

1. Открой [Google Sheets](https://sheets.google.com)
2. Создай новую таблицу (например, "Каталог портфолио")
3. Назови первый лист `Portfolio` (или оставь `Sheet1`)

### 1.2. Создай заголовки столбцов

В первой строке (A1, B1, C1...) создай следующие заголовки:

| A   | B     | C    | D    | E        | F      | G        | H     | I      |
| --- | ----- | ---- | ---- | -------- | ------ | -------- | ----- | ------ |
| id  | title | type | area | duration | budget | location | scope | images |

**Важно:** именно в таком порядке! Это нужно для правильной работы Apps Script.

### 1.3. Добавь 1-2 примера карточек (для теста)

Заполни 2-3 строки с примерами:

```
HS-001 | Реконструкция дома | Строительство | 250 м² | 6 месяцев | 12 млн ₽ | Чеченская Республика | Фундамент, Перепланировка | https://drive.google.com/uc?export=view&id=FILE_ID_1, https://drive.google.com/uc?export=view&id=FILE_ID_2
```

**Примечание:** В колонке `images` будут публичные ссылки на изображения после обработки Apps Script. При заполнении примера можно использовать любые рабочие ссылки на изображения.

### 1.4. Опубликуй таблицу для чтения

1. **Файл** → **Опубликовать в интернете**
2. Выбери:
   - **Веб-страница** или **CSV**
   - **Опубликовать**
3. Скопируй **ID таблицы** из URL:
   ```
   https://docs.google.com/spreadsheets/d/ТВОЙ_ID_ТАБЛИЦЫ/edit
   ```
   Скопируй `ТВОЙ_ID_ТАБЛИЦЫ` — он понадобится позже.

## 📝 ШАГ 2: Создание Google Form

### 2.1. Создай форму

1. Открой [Google Forms](https://forms.google.com)
2. Создай новую форму (например, "Добавить карточку в каталог")

### 2.2. Добавь поля формы

Создай следующие поля (в таком порядке):

1. **Название проекта** (Короткий ответ)

   - Описание: Название проекта или объекта
   - Обязательное поле ✓
   - Пример: "Реконструкция загородного дома"

2. **Тип проекта** (Выпадающий список)

   - Описание: Категория проекта
   - Варианты:
     - Строительство
     - Ремонт
     - Дизайн
   - Обязательное поле ✓

3. **Площадь** (Короткий ответ)

   - Описание: Площадь объекта в квадратных метрах
   - Обязательное поле ✓
   - Пример: "250 м²"

4. **Срок** (Короткий ответ)

   - Описание: Время выполнения работ
   - Обязательное поле ✓
   - Пример: "6 месяцев"

5. **Бюджет** (Короткий ответ)

   - Описание: Стоимость проекта
   - Обязательное поле ✓
   - Пример: "12 млн ₽"

6. **Локация** (Короткий ответ)

   - Описание: Местоположение объекта
   - Обязательное поле ✓
   - Пример: "Чеченская Республика, Курчалоевский район"

7. **Состав работ** (Абзац)

   - Описание: Перечень выполненных работ
   - Обязательное поле ✓
   - Подсказка: "Перечислите через запятую, например: Фундамент, Перепланировка, Новая кровля"

8. **Изображения** (Загрузка файлов)

   - Описание: Фотографии проекта (до/после или готовый результат)
   - Обязательное поле ✓
   - Разрешить только определенные типы файлов: **Изображения** ✓
   - Максимальное количество файлов: **10** (или другое значение по желанию)
   - Максимальный размер файла: **10 МБ** (или другое значение по желанию)

### 2.3. Свяжи форму с таблицей

1. В форме нажми **Ответы** (справа вверху)
2. Нажми **Связать с таблицей** (иконка таблицы)
3. Выбери **Создать новую таблицу** или **Выбрать существующую**
4. Если создаешь новую — назови ее так же, как основную таблицу
5. Если выбираешь существующую — выбери таблицу из Шага 1

**Важно:** Форма автоматически создаст колонки в таблице. Они могут не совпадать с нашими заголовками. Это нормально — Apps Script все исправит.

### 2.4. Скопируй ссылку на форму

1. Нажми **Отправить** (вверху справа)
2. Скопируй ссылку (например, `https://forms.gle/ТВОЯ_ССЫЛКА`)
3. **Сохрани эту ссылку** — ее дашь заказчику

## ⚙️ ШАГ 3: Настройка Apps Script

### 3.1. Открой Apps Script

1. В Google Sheets открой свою таблицу
2. **Расширения** → **Apps Script**
3. Откроется редактор кода

### 3.2. Вставь код обработчика формы

Удали весь код по умолчанию и вставь этот:

```javascript
/**
 * Обработчик отправки формы
 * Автоматически вызывается при каждом новом ответе в форме
 */
function onFormSubmit(e) {
  try {
    const sheet =
      SpreadsheetApp.getActiveSpreadsheet().getSheetByName("Portfolio") ||
      SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();

    // Получаем последнюю строку (только что добавленную формой)
    const lastRow = sheet.getLastRow();

    // Получаем данные из формы
    // ВАЖНО: порядок колонок зависит от порядка полей в форме!
    // Проверь порядок колонок в таблице после первой отправки формы
    const formData = e.values; // Массив значений из формы

    // Определяем индексы колонок (настрой под свою форму!)
    // Обычно Google Forms создает колонки в таком порядке:
    // 0: Timestamp, 1: Название, 2: Тип, 3: Площадь, 4: Срок, 5: Бюджет,
    // 6: Локация, 7: Состав работ, 8: Изображения (загруженные файлы)

    // Если форма создала свои колонки, нужно найти правильные индексы
    // Для этого отправь тестовую форму и посмотри, в каких колонках что появилось

    const title = formData[1] || ""; // Название проекта
    const type = formData[2] || ""; // Тип
    const area = formData[3] || ""; // Площадь
    const duration = formData[4] || ""; // Срок
    const budget = formData[5] || ""; // Бюджет
    const location = formData[6] || ""; // Локация
    const scope = formData[7] || ""; // Состав работ
    const imagesRaw = formData[8] || ""; // Изображения (сырые ссылки из Drive)

    // Генерируем ID
    const existingIds = sheet
      .getRange(2, 1, lastRow - 1, 1)
      .getValues()
      .flat();
    let newId = `HS-${String(existingIds.length + 1).padStart(3, "0")}`;

    // Проверяем, не занят ли ID
    while (existingIds.includes(newId)) {
      const num = parseInt(newId.split("-")[1]) + 1;
      newId = `HS-${String(num).padStart(3, "0")}`;
    }

    // Форматируем scope (если это строка с запятыми)
    const scopeFormatted =
      typeof scope === "string"
        ? scope
            .split(",")
            .map((s) => s.trim())
            .filter(Boolean)
            .join(", ")
        : scope;

    // Записываем данные в правильные колонки
    // Предполагаем, что заголовки уже есть в первой строке
    const headers = sheet.getRange(1, 1, 1, 9).getValues()[0];

    // Находим индексы колонок по заголовкам
    const colIndex = {
      id: headers.indexOf("id") + 1,
      title: headers.indexOf("title") + 1,
      type: headers.indexOf("type") + 1,
      area: headers.indexOf("area") + 1,
      duration: headers.indexOf("duration") + 1,
      budget: headers.indexOf("budget") + 1,
      location: headers.indexOf("location") + 1,
      scope: headers.indexOf("scope") + 1,
      images: headers.indexOf("images") + 1,
    };

    // Записываем данные
    sheet.getRange(lastRow, colIndex.id).setValue(newId);
    sheet.getRange(lastRow, colIndex.title).setValue(title);
    sheet.getRange(lastRow, colIndex.type).setValue(type);
    sheet.getRange(lastRow, colIndex.area).setValue(area);
    sheet.getRange(lastRow, colIndex.duration).setValue(duration);
    sheet.getRange(lastRow, colIndex.budget).setValue(budget);
    sheet.getRange(lastRow, colIndex.location).setValue(location);
    sheet.getRange(lastRow, colIndex.scope).setValue(scopeFormatted);
    sheet.getRange(lastRow, colIndex.images).setValue(publicImages);

    // Удаляем временные колонки, созданные формой (если они есть)
    // Это опционально, можно оставить их для отладки
  } catch (error) {
    // Логируем ошибку
    Logger.log("Ошибка обработки формы: " + error.toString());
    // Можно отправить уведомление на email
    // MailApp.sendEmail('твой@email.com', 'Ошибка формы', error.toString());
  }
}

/**
 * Обрабатывает ссылки на загруженные файлы из Google Drive
 * Конвертирует их в публичные ссылки для изображений
 */
function processImages(imagesRaw) {
  if (!imagesRaw || imagesRaw === "") return "";

  try {
    // Разделяем ссылки (могут быть через запятую, перенос строки или пробел)
    const links = imagesRaw
      .split(/[,\n\r]+/)
      .map((link) => link.trim())
      .filter((link) => link && link.length > 0);

    // Обрабатываем каждую ссылку
    const publicLinks = links
      .map((link) => convertDriveUrlToImage(link))
      .filter((link) => link);

    // Объединяем обратно через запятую
    return publicLinks.join(", ");
  } catch (error) {
    Logger.log("Ошибка обработки изображений: " + error.toString());
    return imagesRaw; // Возвращаем исходную строку при ошибке
  }
}

/**
 * Конвертирует ссылку Google Drive в публичную ссылку на изображение
 */
function convertDriveUrlToImage(driveUrl) {
  if (!driveUrl || driveUrl === "") return "";

  try {
    // Извлекаем ID файла из разных форматов ссылок Drive
    let fileId = null;

    // Формат: https://drive.google.com/file/d/FILE_ID/view
    const match1 = driveUrl.match(/\/d\/([a-zA-Z0-9_-]+)/);
    if (match1) {
      fileId = match1[1];
    }

    // Формат: https://drive.google.com/open?id=FILE_ID
    if (!fileId) {
      const match2 = driveUrl.match(/[?&]id=([a-zA-Z0-9_-]+)/);
      if (match2) {
        fileId = match2[1];
      }
    }

    if (!fileId) {
      // Если не удалось извлечь ID, возвращаем исходную ссылку
      Logger.log("Не удалось извлечь ID файла из ссылки: " + driveUrl);
      return driveUrl;
    }

    // Получаем файл и делаем его публичным
    const file = DriveApp.getFileById(fileId);
    file.setSharing(DriveApp.Access.ANYONE_WITH_LINK, DriveApp.Permission.VIEW);

    // Возвращаем прямую ссылку на изображение
    return `https://drive.google.com/uc?export=view&id=${fileId}`;
  } catch (error) {
    Logger.log(
      "Ошибка конвертации ссылки: " + error.toString() + " | URL: " + driveUrl
    );
    return driveUrl; // Возвращаем исходную ссылку при ошибке
  }
}
```

## 🐛 Решение проблем

### Проблема: Apps Script не срабатывает

**Решение:**

1. Проверь, что триггер настроен правильно
2. Проверь логи в Apps Script: **Выполнения** (часы) → выбери последнее выполнение
3. Убедись, что функция называется `onFormSubmit`

### Проблема: Фото не отображаются

**Решение:**

1. Проверь логи Apps Script (Выполнения → последнее выполнение) — там будут ошибки, если что-то пошло не так
2. Убедись, что файлы в Drive публичные (Apps Script должен сделать их публичными автоматически)
3. Проверь формат ссылки в таблице (должна быть `https://drive.google.com/uc?export=view&id=...`)
4. Открой ссылку в браузере — должно открыться изображение
5. Если файлы не загружаются через форму — проверь настройки поля "Загрузка файлов" (максимальный размер, типы файлов)

### Проблема: Данные в неправильных колонках

**Решение:**

1. Отправь тестовую форму
2. Посмотри, в каких колонках появились данные
3. Обнови индексы в Apps Script (строки `const title = formData[1]` и т.д.)

### Проблема: CORS ошибка при чтении данных

**Решение:**

1. Убедись, что таблица опубликована
2. Используй правильный URL для чтения (см. код в `googleSheets.js`)
3. Если не помогает — используй Apps Script для создания JSON endpoint

## 🔐 Безопасность

**Важно помнить:**

- Ссылка на форму — это единственная "защита"
- Любой, кто узнает ссылку, сможет добавлять карточки
- Для реальной защиты нужен бэкенд с авторизацией
- Но для простого каталога этого обычно достаточно

**Рекомендации:**

- Не публикуй ссылку на форму публично
- Дай ссылку только заказчику
- Периодически проверяй таблицу на спам/мусор
