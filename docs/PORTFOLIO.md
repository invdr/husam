# Блок портфолио (ex/index.html)

`ex/index.html` — это более продвинутый вариант страницы с тем же концептом, но с добавленным блоком **«Портфолио»**.

## Основные отличия от корневого index.html

- Добавлен блок **«Портфолио»** с каруселью проектов
- Навигация в шапке и футере дополнена пунктом **«Портфолио»** (`id="портфолио"`)
- Логика анимаций, валидации форм, калькулятора и контактной формы аналогична корневому `index.html`

`ex/index.html` можно рассматривать как «LAB/EXPERIMENTAL» версию для тестирования идей (карусель, модалки, портфолио).

## Структура данных портфолио

Массив `portfolioData` содержит объекты проектов со следующими полями:

```javascript
{
  id: "HS-001",                    // Уникальный идентификатор проекта
  title: "Реконструкция загородного дома",  // Название проекта
  type: "Строительство",           // Тип работ
  area: "250 м²",                  // Площадь
  duration: "6 месяцев",           // Срок выполнения
  budget: "12 млн ₽",             // Бюджет
  location: "Чеченская Республика, Курчалоевский район",  // Локация
  scope: [                         // Состав работ (массив строк)
    "Фундамент",
    "Перепланировка",
    "Новая кровля",
    "Фасад и благоустройство"
  ],
  beforeImage: "https://...",      // URL изображения "До"
  afterImage: "https://...",       // URL изображения "После"
  testimonial: "Профессиональная команда...",  // Отзыв клиента
  clientName: "Ахмед М."           // Имя клиента
}
```

## Компонент ArrowCarousel

Карусель проектов с адаптивным отображением:

### Функциональность

- **Адаптивное количество карточек на экране**:
  - Мобильные (< 768px): 1 карточка
  - Планшеты (768px - 1024px): 2 карточки
  - Десктоп (> 1024px): 3 карточки
  - Реализовано через хук `usePerView()`

- **Навигация**:
  - Стрелки влево/вправо для переключения слайдов
  - Буллиты-индикаторы снизу для прямого перехода к слайду

- **Карточка проекта**:
  - Сетка изображений «До/После» (2 колонки на мобильных, 4 на десктопе)
  - Основная информация: название, тип, площадь, срок, бюджет, локация
  - Рейтинг (5 звезд)
  - Отзыв клиента
  - Две кнопки:
    - «Подробнее» — открывает модальное окно с детальной информацией
    - «Хочу этот проект» — открывает WhatsApp с информацией о проекте

### Технические детали

- Карусель монтируется через отдельный `ReactDOM.createRoot` в `#carousel-root`
- Используется CSS transform для плавной анимации переключения
- Состояние текущего индекса управляется через `useState`

## Модальное окно проекта

Детальный просмотр проекта через модальное окно:

### Функциональность

- **Открытие**: по клику на кнопку «Подробнее» в карточке проекта
- **Закрытие**:
  - По клику на крестик
  - По клику на overlay (фон)

### Содержимое модального окна

**Левая колонка**:
- Большое изображение «После»
- Детальная информация в виде сетки:
  - Артикул (ID проекта)
  - Тип работ
  - Площадь
  - Срок
  - Бюджет
  - Локация

**Правая колонка**:
- Заголовок «Состав работ» со списком работ из `scope`
- Блок с отзывом клиента
- Две кнопки:
  - «Хочу этот проект» — открывает WhatsApp с информацией о проекте
  - «Задать вопрос» — открывает WhatsApp с запросом консультации

### Технические детали

- Состояние `selectedProject` управляется через `useState` в корневом компоненте
- Модальное окно рендерится условно (`{selectedProject && ...}`)
- Overlay с `bg-black/60` для затемнения фона
- Закрытие по клику на overlay реализовано через `onClick` на `div` с overlay

## Примеры проектов в портфолио

1. **HS-001**: Реконструкция загородного дома (250 м², 6 месяцев, 12 млн ₽)
2. **HS-002**: Ремонт квартиры премиум-класса (120 м², 4 месяца, 5 млн ₽)

## Рекомендации по использованию

- Замените URL изображений `beforeImage` и `afterImage` на реальные фотографии проектов
- Добавьте больше проектов в массив `portfolioData` для более полного портфолио
- Рассмотрите возможность загрузки данных портфолио из внешнего источника (JSON, API) для упрощения управления контентом
