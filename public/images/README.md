# 📸 Инструкция по добавлению фото проектов

## Простая инструкция для не-программистов

### Шаг 1: Создайте папку для проекта

Создайте папку с ID проекта (например, `HS-001`, `HS-002` и т.д.) внутри папки `portfolio`

**Пример:**
```
public/images/portfolio/HS-001/
```

### Шаг 2: Переименуйте фото

Переименуйте все фото проекта в простой формат:
- `01.jpg` - первое фото (главное, будет показываться в карточке)
- `02.jpg` - второе фото
- `03.jpg` - третье фото
- и так далее...

**Важно:**
- Используйте только цифры: `01`, `02`, `03`...
- Расширение файла: `.jpg`, `.jpeg` или `.webp` (рекомендуется `.webp` для лучшего качества)
- Начинайте с `01`, а не с `1`

### Шаг 3: Положите фото в папку

Скопируйте все переименованные фото в папку проекта.

**Пример структуры:**
```
public/images/portfolio/
  HS-001/
    01.jpg  ← главное фото (показывается в карточке)
    02.jpg
    03.jpg
    04.jpg
    05.jpg
  HS-002/
    01.jpg
    02.jpg
    03.jpg
    04.jpg
```

### Шаг 4: Укажите количество фото в коде

Откройте файл `src/data/portfolio.js` и найдите свой проект по ID.

Вместо старого способа:
```javascript
images: [
  "/images/portfolio/HS-001/01.jpg",
  "/images/portfolio/HS-001/02.jpg",
  // ...
]
```

Используйте новый простой способ:
```javascript
{
  id: "HS-001",
  title: "Название проекта",
  // ... другие поля
  imageCount: 5,  // ← просто укажите количество фото!
  imageFormat: "webp",  // ← опционально: формат фото (jpg, webp, png). По умолчанию jpg
}
```

**Все!** Фото автоматически подтянутся.

---

## Примеры

### Проект с 3 фото:
```
public/images/portfolio/HS-013/
  01.jpg
  02.jpg
  03.jpg
```

В коде:
```javascript
{
  id: "HS-013",
  title: "Мой проект",
  imageCount: 3,  // ← 3 фото
  imageFormat: "webp",  // ← если используете .webp файлы
}
```

### Проект с 10 фото:
```
public/images/portfolio/HS-014/
  01.jpg
  02.jpg
  03.jpg
  ...
  10.jpg
```

В коде:
```javascript
{
  id: "HS-014",
  title: "Большой проект",
  imageCount: 10,  // ← 10 фото
}
```

## Советы

1. **Формат фото:** 
   - `.webp` - рекомендуется (лучшее качество при меньшем размере)
   - `.jpg` или `.jpeg` - стандартный формат
   - `.png` - для изображений с прозрачностью
2. **Размер:** Оптимизируйте фото перед загрузкой (рекомендуется 1200-1600px по ширине)
3. **Имена файлов:** Только цифры! Не используйте пробелы, русские буквы или спецсимволы
4. **Порядок:** Первое фото (`01.jpg` или `01.webp`) будет главным - выберите самое красивое
5. **Формат в коде:** Если используете `.webp`, добавьте `imageFormat: "webp"` в данные проекта

---

## Что делать, если нужно указать фото вручную?

Если по какой-то причине нужно указать пути к фото вручную, можно использовать старый способ:

```javascript
{
  id: "HS-015",
  images: [
    "/images/portfolio/HS-015/photo1.jpg",
    "/images/portfolio/HS-015/custom-name.jpg",
  ]
}
```

Но для простоты лучше использовать `imageCount` - это намного проще!
